# App Description
This is a web application built using Flask that helps people find the nearest MBTA station and giving users recommendations on whether the air is a good quality for them to travel on foot to the train station.

## How this App Works
1. The user enters a place name or address
2. This app geocodes it using the Mapbox API
3. This app finds the closest MBTA stop using the MBTA API
4. This app displays the result on a map.
